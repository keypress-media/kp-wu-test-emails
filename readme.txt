== Description ==

Send test emails of WP Ultimo email templates.

== Installation ==

1. Activate the plugin
2. Go to (Network Admin) WP Ultimo > Settings > Emails
3. Scroll down and to the right of the submit button you'll find the "Send Test Email" form.
